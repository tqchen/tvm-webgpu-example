TVM WebGPU Demo

  • You need a browser with web GPU capability. Tested on Chrome Canary
  • End to end time cost includes data copy from/to GPU
  • GPU run benchmarks are runs that only invokes compute without data copy. By comparing run benchmarks with end to end time cost, we can get a sense of the overhead of copy cost.
  • Try to click classify on the same image multiple times, you will notice subsequent runs becomes faster. These number will gives us a sense about what is the time cost once the GPU driver execution stablizes.
